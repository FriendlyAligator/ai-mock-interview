# 🚀 AI Mock Interview Platform

An AI-powered mock interview platform that helps users practice interviews in a realistic, interactive environment with performance feedback.

---

## 🔗 Live Demo
👉 https://interview-ufu7.onrender.com/

---

## 📌 Features

- 🔐 User Authentication (Login / Register / Reset Password)
- 🤖 AI-generated interview questions
- 🧠 Role-based interview selection (Frontend / Backend / Database)
- 💬 Interactive answer submission
- 📊 Performance evaluation with score & feedback
- 📈 Dashboard with interview history and analytics

---

## 🛠️ Tech Stack

**Frontend:**
- React.js

**Backend:**
- FastAPI / Node.js (based on your setup)

**Database:**
- PostgreSQL

**Deployment:**
- Render

**Others:**
- REST APIs
- AI / LLM Integration

## ⚙️ Installation & Setup

### 1. Clone the repository
```bash
git clone https://github.com/FriendlyAligator/ai-mock-interview.git
cd ai-mock-interview

2. Setup Backend
cd backend
pip install -r requirements.txt
uvicorn main:app --reload

3. Setup Frontend
cd frontend
npm install
npm start

📊 How It Works
User logs in or registers
Selects interview role
AI generates questions
User submits answers
System evaluates responses
Results and feedback are displayed
Performance tracked in dashboard
